# 🚗 Car Price Prediction & Optimization

## 💼 Data Science Case Study using CarDekho Dataset

---

## 📌 Project Overview

This project is a **Data Science Case Study** focused on analyzing and predicting used car prices using real-world data from CarDekho.

The goal is to **understand key factors affecting car prices** and build a machine learning model to support **data-driven pricing decisions**.

---

## 🎯 Problem Statement

Used car platforms face challenges in pricing vehicles accurately.

* Overpricing leads to unsold inventory
* Underpricing results in revenue loss

This project aims to **analyze pricing patterns and develop a predictive model** to optimize pricing strategies.

---

## 🎯 Objectives

* Analyze factors influencing car prices
* Perform data preprocessing and feature engineering
* Build a machine learning model (Linear Regression)
* Evaluate model performance
* Generate business insights and recommendations

---

## 📂 Dataset

* Source: CarDekho Used Car Dataset
* Contains real-world car listing data

### Key Features:

* Car Name (Brand extracted)
* Year
* Selling Price (Target)
* Kilometers Driven
* Fuel Type
* Transmission
* Seller Type
* Ownership

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🔧 Project Workflow

1. Data Loading & Understanding
2. Feature Engineering (Brand extraction, Car Age)
3. Data Preprocessing using Pipeline
4. Model Building (Linear Regression)
5. Model Evaluation
6. Prediction on New Data
7. Business Insights & Recommendations

---

## 📊 Model Performance

* R² Score: **0.53**
* Mean Absolute Error: **~1.84 Lakhs**

### Interpretation:

* Model explains ~53% of price variation
* Performance is realistic due to real-world data complexity

---

## 🧠 Key Insights

* Car age significantly reduces resale value
* Diesel cars retain better value
* Automatic cars are priced higher
* Higher km driven lowers price
* Brand strongly impacts pricing

---

## 💼 Business Recommendations

* Implement ML-based pricing suggestions
* Identify overpriced listings
* Highlight undervalued cars
* Adjust pricing based on usage and features

---

## ⚠️ Limitations

* No data on car condition
* No location-based pricing
* Market demand not included

---

## 🚀 Future Improvements

---

## 🏁 Conclusion

This project demonstrates how data science can improve pricing strategies in used car marketplaces and support better decision-making.
